# Traitement des objections (V1)

Objectif : répondre court, revenir à la valeur, proposer une prochaine étape simple.

## “C’est trop cher”

Réponse :
- “Je comprends. Wix est moins cher si vous avez le temps de le faire et de l’optimiser. Ici, vous achetez une machine à demandes : on capte, on qualifie, on répond vite, et on mesure.”
- “Une seule intervention gagnée peut rentabiliser le pack. On le mesure sur 30 jours.”

Question de recadrage :
- “Aujourd’hui, vous recevez combien d’appels/devis par semaine via Google ?”

Prochaine étape :
- “Je vous montre la maquette + le dashboard 10 minutes, et vous décidez ensuite.”

## “J’ai déjà un site / Wix”

Réponse :
- “Parfait. On ne remplace pas forcément, on remplace ce qui ne convertit pas.”
- “Wix est un outil. Moi je vous installe le système complet : formulaire ‘devis 30s’, tri, messages, backoffice, suivi.”

Question :
- “Aujourd’hui, vous savez combien de demandes viennent du site ?”

Prochaine étape :
- “Je fais un audit minute et je vous montre la version conversion.”

## “Je n’ai pas le temps”

Réponse :
- “C’est exactement pour ça que c’est productisé. Vous répondez à un seul formulaire et vous m’envoyez 6 photos. Le reste est géré.”

Prochaine étape :
- “Je vous envoie l’onboarding, ça prend 7 minutes.”

## “Je n’ai pas besoin, je marche au bouche-à-oreille”

Réponse :
- “Très bien, on ne remplace pas le bouche-à-oreille, on le transforme en demandes mesurables.”
- “Le site sert aussi à rassurer : quand quelqu’un vous recommande, il vous cherche sur Google.”

Question :
- “Quand on vous recommande, on tombe sur quoi aujourd’hui ?”

## “Je veux juste un site vitrine”

Réponse :
- “Je peux le faire, mais ça ne sert à rien si ça ne génère pas de demandes. Je préfère vous livrer une page qui convertit + un système de réponse.”

Option :
- “On peut partir sur Starter si vous voulez juste une base, puis évoluer.”

## “Je vais réfléchir”

Réponse :
- “Bien sûr. Pour vous aider à décider, je vous envoie la maquette + le périmètre exact du pack.”

Next step :
- “On se cale un point de 10 minutes [créneau A] ou [créneau B] ?”

## “J’ai eu des mauvaises expériences avec des prestataires”

Réponse :
- “Je comprends. Justement, tout est cadré : périmètre, délai, 1 aller-retour, et vous voyez les demandes + chiffres.”

Preuve :
- “On fait un test : un lead de démo arrive, vous voyez le flux complet.”

## “Et si je veux des modifications”

Réponse :
- “Il y a 1 aller-retour inclus. Ensuite, soit on chiffre, soit c’est couvert par l’abonnement selon la demande.”

## “Et le référencement ?”

Réponse :
- “La base SEO local est intégrée : pages services + zones + contenu clair + vitesse mobile. La priorité V1 c’est conversion + réponse rapide.”

Question :
- “Votre objectif : plus d’appels immédiats ou monter sur des requêtes précises ?”
