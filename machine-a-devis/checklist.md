# Checklist de livraison (V1)

1) Choisir le métier + services activés (3–8)
2) Définir la zone (rayon + villes) + horaires
3) Récupérer logo + 6–12 photos réelles + lien Google Business Profile
4) Définir canaux : téléphone + WhatsApp + email
5) Générer site (Accueil, Services, Zones) + injecter textes + placeholders
6) Ajouter CTA visibles (sticky mobile : Appel / WhatsApp / Devis)
7) Configurer formulaire “Devis en 30 secondes” + champs métier + photos requises
8) Configurer règles : hors zone, urgence, infos manquantes
9) Paramétrer auto-messages (accusé réception, demande photo, proposition créneau, confirmation)
10) Backoffice : statuts + actions 1 clic (appeler, message, RDV, gagné/perdu)
11) Tracking : clic tel, clic WhatsApp, envoi formulaire + source
12) Dashboard : demandes, qualifiées, délai réponse, gagnés/perdus (si suivi)
13) SEO local de base : titles, H1, page services, page zones, NAP cohérent
14) QA mobile : vitesse, lisibilité, boutons, formulaire, confirmation
15) Mise en ligne + test complet (1 lead test) + envoi au client d’un guide 1 page
